====== Computer Security ======
Created jueves 01 diciembre 2011
{{./pasted_image005.png}}

==== Description ====
Course homework and labs will teach students how to find vulnerabilities and how to fix them. The labs are designed to help students practice the principles of secure system design.

==== Provides ====
* how to:
	* design secure systems
	* write secure code
	* find vulnerabilities in code
	* design software systems that limit the impact of security vulnerabilities
* memory safety vulnerabilities
* techniques and tools for vulnerability detection
* sandboxing and isolation
* web security
* network security
* malware detection and defense
* mobile platform security

==== Requires ====
* knowledge of programming languages such as C and C++
* understanding of basic concepts from operating systems such as processes and threads

==== Schedule ====
02/2012

==== Include ====
* lecture videos, which are broken into small chunks, some of these may contain integrated quiz questions
* quizzes that are not part of video lectures
* other assignments
* approximately two hours worth of video content per week
